# emacs-ide

Emacs build with useful utils configured for speed and confortable writing code from console.

# Dependences

- docker
- make

# Build

execute:
```
make build
```
or

```
./build-docker.sh
```


# Install

execute:
```
make install
```

# Run

execute:
```
run-ide.sh <path-to-project>
```
or if you installed emacs-ide
```
ide <path-to-project>
```
